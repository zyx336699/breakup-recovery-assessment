# 抖音小程序上线操作清单

## 当前推荐方案

先用 `web-view` 把已上线的 H5 系统放进抖音小程序。这样可以最快测试抖音流量、直播挂载和转化数据。

## 需要你提供或完成

1. 抖音开放平台账号。
2. 已创建的小程序 AppID。
3. 小程序主体和类目选择。
4. 抖音后台 `web-view` 业务域名校验文件。
5. 抖音开发者工具登录状态。

## 我已经准备好的内容

- `douyin-miniapp/`：抖音小程序项目壳。
- 首页 `web-view` 指向公网系统。
- URL 自动带来源参数：`source=douyin_miniapp&utm_source=douyin_miniapp`。
- 这样后台“用户来源分布”可以区分抖音小程序来的用户。

## 业务域名配置

在抖音小程序后台找到业务域名或 web-view 域名配置，添加：

```text
https://breakup-recovery-assessment.onrender.com
```

如果平台要求下载校验文件，把文件发给我，我会把它放到网站根目录并重新部署。

## 审核说明建议

小程序简介可写：

```text
分手挽回自测评估系统是一款面向亲密关系修复场景的自测工具，用户通过选择关系阶段、分手原因、联系状态、情绪稳定度等信息，生成阶段判断、避坑提醒和行动建议。系统不承诺必然复合，主要帮助用户理性复盘关系、降低冲动联系和错误沟通。
```

## 风险点

- 如果抖音要求必须原生页面，`web-view` 壳可能审核不通过。
- 如果个人主体不支持相关类目或 web-view 能力，需要换企业主体或改原生小程序。
- 情感挽回类表达要避免“保证复合”“百分百挽回”“付费必成”等承诺。
