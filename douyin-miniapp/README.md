# 抖音小程序接入说明

这是“分手挽回自测评估系统”的抖音小程序壳项目，第一版通过 `web-view` 承载已经部署好的 H5 系统：

https://breakup-recovery-assessment.onrender.com

## 为什么先用 web-view

- 现有系统已经有完整测评、后台统计、DeepSeek 分析、老师点击率和报告复制功能。
- 小程序壳可以最快上线验证流量，不需要重写全部表单和报告逻辑。
- 后续如果审核或体验要求更高，再逐步改成原生小程序页面。

## 你需要准备

1. 抖音开放平台账号，并创建小程序，拿到 AppID。
2. 下载并登录抖音开发者工具。
3. 在抖音小程序后台配置 `web-view` 业务域名：
   - `https://breakup-recovery-assessment.onrender.com`
4. 如果后台要求放置校验文件，把校验文件名和内容发给 Codex，再加入到网站根目录并重新部署。

## 开发者工具操作

1. 打开抖音开发者工具。
2. 选择导入项目。
3. 项目目录选择本文件夹：`douyin-miniapp`。
4. 把 `project.config.json` 里的 `appid` 替换成你的小程序 AppID。
5. 预览确认能打开测评系统。
6. 上传代码。
7. 到抖音开放平台提交审核。

## 注意

- web-view 必须使用已配置的 HTTPS 业务域名。
- 如果个人主体或类目不支持 web-view，需要改成原生小程序版本。
- DeepSeek key 仍然只在后端，不要放进小程序代码。
- 抖音审核时，建议准备清楚产品说明：这是“情感关系自测与行动建议工具”，避免承诺“必然复合”或夸大结果。
